HTML Tags - recherche

Meta Element

Das meta-Element steht im head und liefert Metadaten (Informationen über die Seite die nicht sichtbar sind).

Es gehöhrt zu der Kategorie Metadaten-inhalt. Durch die verschiedenen Attribute kann es unterschiedliche Sufgaben erledigen.

    Beispiele:

  • content = der Wert der zu zbs. namen gehört
  • http-equiv = gibt die Info zur benutzung der Seite
  • name = Die Art von Info die du speichern willst
  • media = für welches Medium oder welche Situation die Metadaten gelten sollen
  • charset = gibt die Zeichenkodierung ab (damit es Umlaute ect. darstellen kann)

Link Element

Das Link element gehöhrt auch zu den Metadaten-Inhalt. Es verbindet die aktuelle Seite mit anderen Ressourcen und externen Dateien

Es wird in das Head geschrieben und hat auch verscheidene Attribute mit verschiedenen Aufgaben

    Beispiele:

  • href = gibt den Pfad zur ressource an
  • rel = Art der Beziehung zwischen der aktuellen Seite und der Ressource
  • type = welcher Dateityp gerade verknüpt wird
  • media = gibt den Medientyp der Ressource an

Script Element

Mit Script kannst du einen Javascript code direkt in einer HTML seite einfügen oder die seite mit der javascript datei verknüpfen.

Man schreibt den Script entweder in den head oder am ende vom Body. Diesese Element hat wieder unterschiedliche Attribute.

    Beispiele:

  • src = Pfad zur externen Javascript datei
  • type = gibt den typ des Scripts an(zbs. text/javascript)
  • async = das script wird asynchron geladen
  • defer = Skript wird nach dem Laden des HTMLs, aber vor dem DOMContentLoaded ausgeführt
  • crossorigin= gibt an wie browser mit anderen externen skripten von anderen domains umgehen soll
  • integrity = prüft ob die Datie unverändert ist

Ul element

Das Ul element gehört zum Flow Content(Sichtbarere Inhalt)

Es wird benutzt um eine ungeordnete Liste zu erstellen. Die einzelnen Punkte der Liste werden mit dem Li element erstellt.

Es gibt ein paar Attribute die sind aber nicht so wichtig.

Ol element

Das Ol element gehört auch zum Flow Content

Es wird benutzt um eine geordnete Liste zu erstellen. Die einzelnen Punkte der Liste werden auch mit dem Li element erstellt.

Anders als beim ul ist die Reihenfolge der Punkte hier wichtig. Um die punkte zu nummerieren braucht man folgende Attribute

    Beispiele:

  • type = gibt den typ der nummerierung an(zbs. Zahlen, Buchstaben)
  • start = legt die start nummer der liste fest
  • reversed = die liste wird in umgekehrter Reihenfolge angezeigt
  • class/id/style= Standard-HTML-Attribute → für CSS oder JS

Li Element

Das Li element gehört auch zum Flow Content

Es wird benutzt um ein Eintrag in einer Liste (geordneten oder ungeordneten Liste) zu ermöglichen.

Es gibt 2 Attribute.

  • value = legt den Wert des Listeneintrags fest (nur bei ol element)
  • class/id/style= Standard-HTML-Attribute → für CSS oder JS
