HTML Tags - recherche

Meta Element

Das meta-Element steht im head und liefert Metadaten (Informationen über die Seite die nicht sichtbar sind).

Es gehöhrt zu der Kategorie Metadaten-inhalt. Durch die verschiedenen Attribute kann es unterschiedliche Sufgaben erledigen.

Beispiele:

  • content = der Wert der zu zbs. namen gehört
  • http-equiv = gibt die Info zur benutzung der Seite
  • name = Die Art von Info die du speichern willst
  • media = für welches Medium oder welche Situation die Metadaten gelten sollen
  • charset = gibt die Zeichenkodierung ab (damit es Umlaute ect. darstellen kann)

Link Element

Das Link element gehöhrt auch zu den Metadaten-Inhalt. Es verbindet die aktuelle Seite mit anderen Ressourcen und externen Dateien

Es wird in das Head geschrieben und hat auch verscheidene Attribute mit verschiedenen Aufgaben

Beispiele:

  • href = gibt den Pfad zur ressource an
  • rel = Art der Beziehung zwischen der aktuellen Seite und der Ressource
  • type = welcher Dateityp gerade verknüpt wird
  • media = gibt den Medientyp der Ressource an

Script Element

Mit Script kannst du einen Javascript code direkt in einer HTML seite einfügen oder die seite mit der javascript datei verknüpfen.

Man schreibt den Script entweder in den head oder am ende vom Body. Diesese Element hat wieder unterschiedliche Attribute.

Beispiele:

  • src = Pfad zur externen Javascript datei
  • type = gibt den typ des Scripts an(zbs. text/javascript)
  • async = das script wird asynchron geladen
  • defer = Skript wird nach dem Laden des HTMLs, aber vor dem DOMContentLoaded ausgeführt
  • crossorigin= gibt an wie browser mit anderen externen skripten von anderen domains umgehen soll
  • integrity = prüft ob die Datie unverändert ist

Ul element

Das Ul element gehört zum Flow Content(Sichtbarere Inhalt)

Es wird benutzt um eine ungeordnete Liste zu erstellen. Die einzelnen Punkte der Liste werden mit dem Li element erstellt.

Es gibt ein paar Attribute die sind aber nicht so wichtig.

Ol element

Das Ol element gehört auch zum Flow Content

Es wird benutzt um eine geordnete Liste zu erstellen. Die einzelnen Punkte der Liste werden auch mit dem Li element erstellt.

Anders als beim ul ist die Reihenfolge der Punkte hier wichtig. Um die punkte zu nummerieren braucht man folgende Attribute

Beispiele:

  • type = gibt den typ der nummerierung an(zbs. Zahlen, Buchstaben)
  • start = legt die start nummer der liste fest
  • reversed = die liste wird in umgekehrter Reihenfolge angezeigt
  • class/id/style= Standard-HTML-Attribute → für CSS oder JS

Li Element

Das Li element gehört auch zum Flow Content

Es wird benutzt um ein Eintrag in einer Liste (geordneten oder ungeordneten Liste) zu ermöglichen.

Es gibt 2 Attribute.

  • value = legt den Wert des Listeneintrags fest (nur bei ol element)
  • class/id/style= Standard-HTML-Attribute → für CSS oder JS

P element

Das P element gehört auch zu dem Flow Content

Es wird benutzt um einen Absatz zu erstellen. Dieses Element hat wieder 2 Attribute

  • class/id/style= Standard-HTML-Attribute → für CSS oder JS
  • align = legt die Ausrichtung des Textes fest (links, rechts, zentriert, Blocksatz)

Br Element

Das Br element gehört auch zum Flow Content

Es wird benutzt um einen Zeilenumbruch zu erstellen. Es 2 attribute.

  • class/id/style= Standard-HTML-Attribute → für CSS oder JS
  • clear = Dieses Attribut ist veraltet

B Element

Das B element gehört zum phrasing content

Es wird benutzt um Text optisch fett darzustellen. Es hat 1 Attribut(=class/id/style)

Strong Element

Das Strong element gehört auch zum phrasing content

Es wird benutzt um Text semantisch korrekt(Browser weiß das es wichtig ist) als wichtig zu betonen. Es hat 1 Attribut(=class/id/style)

i Element

Das I element gehört auch zum phrasing content

Es wird benutzt um Text optisch kursiv darzustellen. Es hat 1 Attribut(=class/id/style)

Em Element

Das Em element gehört auch zum phrasing content

Es wird benutzt um Text semantisch korrekt als hervorgehoben darzustellen. Es hat die globalen Attribute

U Element

Das U element gehört auch zum phrasing content

Es wird benutzt um Text unterstrichen darzustellen. Es hat die globalen Attribute

Small Element

Das Small element gehört auch zum phrasing content

Es wird benutzt um Text kleiner darzustellen auch semantisch. Es hat die globalen Attribute

a Element

Das A element gehört bei einer bestimmten bedingung zum phrasing content

Es wird benutzt um Hyperlinks zu erstellen. Es hat mehrere Attribute

Beispiele:

  • href = gibt die Zieladresse des Links an
  • target = legt fest, wo der Link geöffnet wird (_blank, _self, _parent, _top)
  • rel = beschreibt die Beziehung zwischen der aktuellen Seite und der verlinkten Seite
  • title = zusätzlicher Informationstext, der angezeigt wird
  • download = löst einen download der verlinkten datei aus

img Element

Das Img element gehört zum flow content,phrasing content und embedded content

Es wird benutzt um Bilder in eine HTML Seite einzufügen. Es hat mehrere Attribute

Beispiele:

  • src = Pfad zum Bild
  • alt = beschreibung des Bildes
  • width/height= legt die Breite und Höhe des Bildes in pixel fest
  • title = zusätzlicher Informationstext, der angezeigt wird
  • loading = legt fest, ob das Bild sofort oder erst beim Scrollen geladen wird (lazy/eager)
  • crossorigin = CORS-Handling für externe Bilder

table Element

Das Table element gehört zum flow content

Es wird benutzt um tabellen zu erstellen. Es hat die globalen Attribute, da die anderen veraltet sind

Tr Element

Das Tr element gehört zum flow content

Es wird benutzt um eine Tabellenzeile zu erstellen. Es hat die globalen Attribute,da die anderen veraltet sind

Th Element

Das Th element gehört zum flow content/table content

Es wird benutzt um eine Tabellenüberschrift zu erstellen. Es hat mehrere Attribute

Beispiele:

  • scope = legt den Geltungsbereich der Überschrift fest (row, col, rowgroup, colgroup)
  • colspan = gibt an, über wie viele Spalten die Überschrift erstreckt
  • rowspan = gibt an, über wie viele Zeilen die Überschrift erstreckt

Td Element

Das Td element gehört zum flow content/table content

Es wird benutzt um eine Tabellenzelle zu erstellen. Es hat mehrere Attribute

Beispiele:

  • colspan = gibt an, über wie viele Spalten die Zelle erstreckt
  • rowspan = gibt an, über wie viele Zeilen die Zelle erstreckt
