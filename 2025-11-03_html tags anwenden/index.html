table Element

Das Table element gehört zum flow content

Es wird benutzt um tabellen zu erstellen. Es hat die globalen Attribute, da die anderen veraltet sind

Tr Element

Das Tr element gehört zum flow content

Es wird benutzt um eine Tabellenzeile zu erstellen. Es hat die globalen Attribute,da die anderen veraltet sind

Th Element

Das Th element gehört zum flow content/table content

Es wird benutzt um eine Tabellenüberschrift zu erstellen. Es hat mehrere Attribute

Beispiele:

  • scope = legt den Geltungsbereich der Überschrift fest (row, col, rowgroup, colgroup)
  • colspan = gibt an, über wie viele Spalten die Überschrift erstreckt
  • rowspan = gibt an, über wie viele Zeilen die Überschrift erstreckt

Td Element

Das Td element gehört zum flow content/table content

Es wird benutzt um eine Tabellenzelle zu erstellen. Es hat mehrere Attribute

Beispiele:

  • colspan = gibt an, über wie viele Spalten die Zelle erstreckt
  • rowspan = gibt an, über wie viele Zeilen die Zelle erstreckt

Beispiel für eine Tabelle

Überschrift 1 Überschrift 2
Zelle 1 Zelle 2
Zelle 3 Zelle 4

img Element

Das Img element gehört zum flow content,phrasing content und embedded content

Es wird benutzt um Bilder in eine HTML Seite einzufügen. Es hat mehrere Attribute

Beispiele:

  • src = Pfad zum Bild
  • alt = beschreibung des Bildes
  • width/height= legt die Breite und Höhe des Bildes in pixel fest
  • title = zusätzlicher Informationstext, der angezeigt wird
  • loading = legt fest, ob das Bild sofort oder erst beim Scrollen geladen wird (lazy/eager)
  • crossorigin = CORS-Handling für externe Bilder

Form/Input

Das form-Element ist ein Container, der Eingabefelder, Buttons usw. enthält und Daten an einen Server sendet

Das input-Element erstellt ein einzelnes Eingabefeld (z. B. Text, Zahl, Passwort, Checkbox usw.).

Attribute für form:

  • action=Ziel-URL, an die die Formulardaten gesendet werden.
  • method=Bestimmt, wie Daten gesendet werden: get oder post.
  • enctype=Gibt das Datenformat beim Senden an (z. B. für Datei-Uploads).
  • target=Wo die Antwort angezeigt wird (_self, _blank, _parent, _top).
  • autocomplete=Schaltet Autovervollständigung ein oder aus (on / off).
  • novalidate=Deaktiviert die Browser-Überprüfung der Eingaben.
  • Globale Attribute

Attribute für input:

  • type=Ohne dem weiß der Browser nicht, welche Art von Feld du willst (Text, Passwort, E-Mail, Zahl …).
  • name=Wird gebraucht, damit beim Absenden ein Wert gespeichert oder gesendet wird.
  • value=Gibt den aktuellen oder voreingestellten Wert an. Wichtig bei Buttons oder Voreinstellungen.
  • placeholder=Zeigt Hilfetext an, wenn das Feld leer ist (nützlich für Benutzerfreundlichkeit).
  • required=Macht das Feld Pflicht, Benutzer können das Formular nicht abschicken, wenn es leer ist.

Ol element

Das Ol element gehört auch zum Flow Content

Es wird benutzt um eine geordnete Liste zu erstellen. Die einzelnen Punkte der Liste werden auch mit dem Li element erstellt.

Anders als beim ul ist die Reihenfolge der Punkte hier wichtig. Um die punkte zu nummerieren braucht man folgende Attribute

Beispiele:

  • type = gibt den typ der nummerierung an(zbs. Zahlen, Buchstaben)
  • start = legt die start nummer der liste fest
  • reversed = die liste wird in umgekehrter Reihenfolge angezeigt
  • class/id/style= Standard-HTML-Attribute → für CSS oder JS

Li Element

Das Li element gehört auch zum Flow Content

Es wird benutzt um ein Eintrag in einer Liste (geordneten oder ungeordneten Liste) zu ermöglichen.

Es gibt 2 Attribute.

  • value = legt den Wert des Listeneintrags fest (nur bei ol element)
  • class/id/style= Standard-HTML-Attribute → für CSS oder JS

Beispiel für eine ol liste

  1. Apfel
  2. Banane
  3. Kirsche
  4. Traube
